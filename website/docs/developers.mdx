## APIs

<Typography variant="body2" sx={{ fontSize: "1.5em" }}>
  Switchboard's APIs provide a client interface to interact with Switchboard V2
  on-chain.
</Typography>

| Package                                                           | Description                                                   |
| ----------------------------------------------------------------- | ------------------------------------------------------------- |
| [Typescript](https://docs.switchboard.xyz/api/ts)                 | Typescript client to interact with Switchboard V2.            |
| [Typescript **_Lite_**](https://docs.switchboard.xyz/api/ts-lite) | Typescript "Lite" client to deserialize aggregator accounts   |
| [Python](https://docs.switchboard.xyz/api/py)                     | Python client to interact with Switchboard V2.                |
| [Rust](https://docs.rs/switchboard-v2/latest/switchboard_v2/)     | Rust client to interact with Switchboard V2.                  |
| [CLI](/api/cli)                                                   | Command Line Interface (CLI) to interact with Switchboard V2. |
